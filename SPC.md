# SPC File Format (WIP)
4 Magic (SCRP)  
4 File size  
4 Magic (CODE)  
4 Code size  
6 0x00 0x04 0x00 0x00 0x00 0x00  
[...code...]  
3 0x8F 0x00 0x00  
4 Magic (FUNC), Optional  
4 Func size  
[...func...]  
4 Magic (TERM)
